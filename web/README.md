# Minima Web

Minima LMS web interface built with SolidJS and TypeScript

## Quick Start

```bash
git clone https://github.com/cobel1024/minima && cd minima/web
npm install
docker compose up -d
```

Access: [http://localhost:5173](http://localhost:5173)

## Requirements

- Node.js 22+
- Docker

## Tech Stack

- SolidJS, TypeScript, daisyUI

Docs: [https://cobel1024.github.io/minima-docs/](https://cobel1024.github.io/minima-docs/)

## Development

```bash
npm run upgrade
npm run openapi-ts
npm run i18next-extract
```
